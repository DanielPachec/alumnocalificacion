Sistema para asignar 5 calificaciones a alumnos, mostrar lista de aprobados y reprobados, 
en caso de que el alumno sea reprobado mostrar botón para enviar mensaje a su responsable.
1.- En la carpeta respaldo bd se encuentra el archivo escuela.sql (respaldo de la base de datos para el sistema).
2.-En servidor local insertar la ruta /public/alumno para ingresar al listado de alumnos que 
se encuentran registrados actualmente. Contiene acciones para ingresar un nuevo alumno, editarlo, eliminarlo 
y asignarle las 5 calificaciones.
3.- En la ruta /public/calificación se muestra el listado de las calificaciones que ya fueron 
asignadas a los alumnos, tiene opciones para editar las calificaciones, o eliminar el registro de ellas. 
También se muestran los botones para observar solo los alumnos aprobados y alumnos reprobados, al dar clic 
en el botón de alumnos reprobados se mostrara un listado de alumnos con promedio menor a 6 seguido de 
un botón para enviar un mensaje a su responsable.
